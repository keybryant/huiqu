III Passive Information Collection

When browsing the website, some information can be passively collected by various techniques and methods instead of being provided by you actively. These techniques and methods include IP address, cookies, internet tags, navigation data collection etc.

This website uses IP address. An IP address is a number provided by internet supplier to you to visit internet. The IP address is not your identification information, because in most cases the IP address is dynamic (changes every time you use internet) but not static (used by a PC of specified user). We use IP address to confirm problems of our server, report some aggregated information, decide fast route to connect your PC to our website, manage and improve our website.

Cookie is bit information that is sent by an internet website to your web browser to help the website to memorize your information and preference.

Session cookies is temporary bit information that will be deleted once your close your browser or computer. Session cookies are used to improve website navigation and collect some aggregated information. This website uses session cookies.

Persistent cookies is a permanent information stored in hard disks of your computer forever unless you delete them. The permanent information is stored in your computer with these purposes including finding information you provided before (for example, password), help decide useful web pages to visitors and personalize the website according to your preference. This website uses persistent cookies. Internet tags, famous as gif file, smaller than cookie, define service information of website and are related to IP address and computer browsing ways of different visitors. This website uses internet tags. Tags have been placed on web ads that take people to the site and on different pages of the site. These tags show how many times a page was clicked and what information was visited. However we cannot collect or search personal information via these tags.

Navigation data (log files, server logs and "click" data) are used for system management, website content improvement, marketing purposes and to enhance information exchange with visitors. This website uses navigation data.
